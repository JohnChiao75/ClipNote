# ClipNote

> 一个跨平台的笔记和日记应用！

## 特性
- ~~支持 Markdown 语法高亮~~
- ~~支持多种主题和自定义主题~~
- 支持密码保护笔记
- ~~跨平台支持 Windows、macOS、Linux 和 Android~~
- ~~支持多种导出格式（PDF、HTML、Markdown）~~
- 支持在本地存储笔记数据，保护隐私

## 安装

下载 [GitHub Release](https://github.com/ILoveScratch2/clipnote/releases) 上的最新发布版本，适用于你的操作系统。

镜像站：
- [ILS Open Source Mirror](https://mirror.ilovescratch.dpdns.org)

### Windows / macOS / Linux

下载 `.zip` 后解压（不要直接双击压缩包里的可执行文件），解压后运行。

## 使用

打开应用后，你可以创建一个新的笔记本，或者导入已有的笔记本。应用支持多种笔记格式，并且可以通过密码保护你的笔记内容。
